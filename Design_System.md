## Design System (use ONLY these values — never approximate)

Color \& Tonal Depth:

Our color palette is rooted in a foundation of trust (`primary: #005c9b`) and sophisticated depth (`on\_primary\_fixed: #001d36`).

Primary navy: #1B3A5C (sidebar backgrounds, dark elements)
Primary blue: #2E75B6 (buttons, links, active states)
Accent blue: #1F6099 (hover states, secondary headings)
Light blue: #D6E4F0 (callout backgrounds, focus rings)
Mid blue: #BDD7EE (dividers, borders)
White: #FFFFFF
Off-white: #F8FAFC (page background)
Gray-700: #344054 (body text)
Gray-900: #101828 (headings)
Success: #12B76A | Warning: #F79009 | Error: #F04438

Typography:

* Mobile: use system fonts. Heading sizes: 28px (h1), 22px (h2), 17px (h3). Body: 15px.
* Web: Use 'Inter' (Google Fonts). Heading: 28px/24px/20px. Body: 14px.
